# hackathon_2021

## 2021년 컴퓨터학부 해커톤 레포

팀명 : 옹심이 칼국수

팀장 : 오영선

팀원 : 김유진, 송혜경, 김다운

#### 대회 주제 : COVID-19로 변화된 대학 생활을 개선할 수 있는 창의적이고 혁신적인 소프트웨어 자유 개발

##### 옹심이 - 경북대학교 재학생 간 도움 및 심부름 대행 플랫폼
- 사용 대상 : 경북대학교 재학생
- 심부름, 물품 대여 등 도움이 필요할 때 같은 경북대 재학생에게 부탁하고 코인을 지불하는 (심부름 대행)서비스 제공 어플리케이션


#### 제작 목적

- 제작 배경 : 팀원이 대부분 자취생, 기숙사 생이라 아플 때 약을 사다준다거나, 갑자기 비가 올 때 우산을 빌려야 하는 등 도움이 필요한 일이 종종 있었던 경험을 바탕으로 믿을 수 있는 같은 경북대학교 재학생들이 서로 필요한 도움이나 물품을 제공하는 어플리케이션을 생각하게 됨.
